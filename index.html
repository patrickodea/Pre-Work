CSS

  • inline CSS, internal CSS, external CSS
  • inline CSS used to style attributes directly in the HTML file (style=color:red)
  • internal CSS rules only apply to a single page (this is added in the head section)
  • external CSS links style directly to another file (most common method)
  • * applys style to all elements on the page
  • You can combine multiple selectors by using a , (header, footer will apply the code to both sections)
  • link element is used to connect external files
  • rel specifies the relationship between the two documents
  • href is the URL of the external resource
  • . before the card element affects all cards. This can be done for any element
