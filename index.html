HTML

  • Define Doc type
  • Title is what displays for the browser ID
  • Head is used to display information about the webpage
  • The body is the visible content on webpage
  • h2 h3 h4.... is each tier of header
  • 3 different types of lists ul li ol

CSS

  • inline CSS, internal CSS, external CSS
  • inline CSS used to style attributes directly in the HTML file (style=color:red)
  • internal CSS rules only apply to a single page (this is added in the head section)
  • external CSS links style directly to another file (most common method)
  • * applys style to all elements on the page
  • You can combine multiple selectors by using a , (header, footer will apply the code to both sections)
  • link element is used to connect external files
  • rel specifies the relationship between the two documents
  • href is the URL of the external resource
  • . before the card element affects all cards. This can be done for any element

Git

  • git add [file] add a file as it looks now to your next commit (stage)
  • git commit -m “[descriptive message]” commit your staged content as a new commit snapshot
  • git pull fetch and merge any commits from the tracking remote branch
  • git push [alias] [branch] Transmit local branch commits to the remote repository branch
  • Example: git add -A git commit -m “updated project README file” git pull origin main git push origin feature/project-README

JavaScript

  • Declare a variable using var =
  • Strings are characters used to represent text
  • You can use Chrome DevTools for debugging/testing if needed (Right click > Inspect)
  • Use if statements for conditional statements. Will only execute if the statement meets a condition.
  • Use else if for more conditons.
  • === is used for exact comparison. This provides a boolean value of either true or false.
  • An array is a single variable used to hold a group of data.
  • for loops cycle through arrays to perform all tasks within it
  • A function is a set of instructions that tells the computer how to perform a certain task "function functionname()" To execute the function int the function by adding it to the code "functionname();"
